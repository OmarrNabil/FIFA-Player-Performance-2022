# FIFA World Cup 2022 Player Performance Analysis

## Overview
This project analyzes player performance in the FIFA World Cup 2022 using machine learning models.

## Objective
The goal is to explore whether player performance can be predicted using:
- Goals
- Assists
- Minutes Played

## Dataset
- Source: FootyStats
- 822 players
- 271 features

## Methods
- Data cleaning and preprocessing
- Linear Regression
- Decision Tree
- Random Forest

## Results
Linear Regression performed best (R² ≈ 0.04), but overall prediction accuracy was low.

## Conclusion
Basic statistics alone are not enough to accurately predict player performance.

## Ethics
This project highlights potential bias when evaluating players using only offensive statistics.
